## `— THE CONCEPT`

**AIR/23** é um editorial digital de página única sobre o legado Air Jordan — do 01 ao 04.
Construído como uma revista premium interativa, vive no cruzamento entre cultura, esporte e design.

Sem framework. Sem build step. Sem dependências.
Abre o `index.html`. Pronto.

## `— STACK`

```
REACT 18     —  CDN, sem bundler
BABEL        —  transpilação JSX no browser
CSS3         —  custom properties, grid, blend-modes
VANILLA JS   —  requestAnimationFrame, IntersectionObserver
```

**Arquitetura:**

```
index.html        ← entry point + tweaks defaults
app.jsx           ← todos os componentes React
sneakers.jsx      ← dados dos capítulos + Sneaker component
tweaks-panel.jsx  ← painel de controle flutuante
styles.css        ← ~1800 linhas, theming por custom properties
```
